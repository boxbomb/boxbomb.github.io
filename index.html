在此面板上有多个屏幕，大屏幕从上到下依次是D1-D3，小屏幕从上到下依次为S1-S3，屏幕S1，永远显示当前加密步骤，除第一页

在第一页顶部有一个加密单词，点击换页键可以更换页面，每一个页面都为前一页面所输出进行加密，可以再下面找到方法

当你得到解谜单词时，可以有键盘输入，并提交

加密方法可能告诉你：通过获得布尔正则表达值来创建密钥，具体方式如下：

获取单词并删除重复字母

将第一个布尔表达字母转为条件，第二个布尔字母转为所需数字，通过需求，判断此布尔表达式是否正确

取一个字母表删除单词中所包含字母

如果布尔表达式为真，请将字母表放在单词后面，否则放在单词前面

A数字为偶数
B数字为奇数
C数字为2的倍数
D数字为质数
E数字的倍数为25
F数字为平方数
G数字为立方数
H数字% 4 = 2,4
I数字共享与循环到1的奇偶校验（16-8-4-2-1-5,5 和 16 奇偶不同，所以为否）

关于凯撒密码(CC,1)

最经典的密码

取加密字符串，并取屏幕D1上的数字N，将加密字符串的每一个字母，在字母表中向前推进N+1个字母，那个字母为新加密字母

示例1：

数字N：21
加密字符串：WPPWYD
WPPWYD -> ATTACH
解密完成：ATTACH

示例2：

数字N：5
加密字符串：GRVNGKZ
GRVNGKZ -> ALPHABET
解密完成：ALPHABET

关于ATBASH密码(AC,1)

最经典的密码之二

取加密字符串，将每一个字母转为字母位置，并从27中减去，将新的数字转为字母，这位你的新解密字母

示例1：

加密字符串：ZJJOVH
ZJJOVH -> APPLES
解密完成：APPLES

示例2：

加密字符串：HJFZIV
HJFZIV -> SQUARE
解密完成：SQUARE

关于单字母替换密码(SC,1)

当然字符串的魔力大于一切

通过将屏幕D1和S1的关键词和布尔表达式的结果，创建字符串，其中条件为第一个字母，第二个字母为字母位置同等的数字

将一个字母表放在字符串下面，通过将加密字符串的每个字符，转为它正下方的字母即为新加密字母

示例1：

加密字符串：ESJJG
关键词：LAUGH
布尔表达式：AQ
AQ -> FASLE -> BCDEFIJKMNOPQRSTVWXYZLAUGH
BCDEFIJKMNOPQRSTVWXYZLAUGH
ABCDEFGHIJKLMNOPQRSTUVWXYZ
解密完成：DOGGY

示例2：

加密字符串：SHGJ
关键词：MAGA
布尔表达式：AQ
CT -> TRUE -> MAGBCDEFHIJKLNOPQRSTUVWXYZ
MAGBCDEFHIJKLNOPQRSTUVWXYZ
ABCDEFGHIJKLMNOPQRSTUVWXYZ
解密完成：SICK

关于滑动解锁密码(HC,1)

你不希望你忘记密码吧？也是平衡公平密码的简化版本！

通过将屏幕D1和S1的关键词和布尔表达式的结果，创建字符串，其中条件为第一个字母，第二个字母为字母位置同等的数字

取字母表放在字符串下面，并将加密单词每个字母分成两份，对于每一份，取屏幕D2上对应的字母，将字母表的第一个字母平移为这个字母

取每一对的第一个字母在字符串中找到，第二个字母在字母表中找到

如果两个字母在同一列，保持不动

否则取每个字母在同行但在里一个字母列的字母

示例1：

加密字符串：KNFZOO
关键词：WRAPPED
屏幕D2:RVX
布尔表达式：EY
EY -> TRUE -> WRAPEDBCFGHIJKLMNOQSTUVXYZ
WRAPEDBCFGHIJKLMNOQSTUVXYZ
ABCDEFGHIJKLMNOPQRSTUVWXYZ
第一对
WRAPEDBCFGHIJKLMNOQSTUVXYZ
RSTUVWXYZABCDEFGHIJKLMNOPQ
KN->VE
第二对
WRAPEDBCFGHIJKLMNOQSTUVXYZ
VWXYZABCDEFGHIJKLMNOPQRSTU
FZ->ED
第三对
WRAPEDBCFGHIJKLMNOQSTUVXYZ
XYZABCDEFGHIJKLMNOPQRSTUVW
OO->OO
解密完成：VEEDOO
